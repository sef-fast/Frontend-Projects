## Project 4A — Team Members Card (Initial)
![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

**Where it's used:** Company websites, startup landing pages,
agency portfolios, SaaS about pages

**Problem it solves:** Presents a team in a clean, responsive
grid layout that works on all screen sizes

**Techniques used:**
- CSS Grid with auto-fit and minmax
- Flexbox inside grid cards
- Avatar image with aspect-ratio
- Card hover lift animation
- Shadow transitions
- object-fit: cover on images
