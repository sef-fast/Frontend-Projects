**Where it's used:** SaaS websites, tech startups, digital
agencies, any company needing a complete team page

**Problem it solves:** A production-ready dark-themed team
page with sticky navbar, animated hero, stats section,
filter bar, featured card, and footer

**Techniques used:**
- Full CSS token system (spacing, typography, radius, transitions)
- Sticky navbar with z-index layering
- CSS Grid for hero stats section
- backdrop-filter blur on stat cards
- Featured card with grid-column: span 2
- Underline animation on links using ::after pseudo-element
- Linear gradient hero background
- CSS Variables used for every value consistently
- border on avatar image for accent ring
